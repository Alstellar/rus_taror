# Правила проекта

1. Все сообщения, отправляемые ботом, должны быть на русском языке.
2. При работе с картами Таро использовать только классический набор Райдера-Уэйта.
3. Все комментарии в проекте должны быть на русском языке.
4. У каждой функции должно быть краткое описание ее функционала на русском языке.
5. Придерживайся четкой структуры и модульности проекта. Разделяй проект на соответствующие блоки и модули.
6. Каждый модуль должен отвечать только за свою задачу. Старайся не допускать большого размера модулей - не более 500 строк каждый файл.
7. Перед внесением изменения в проекте ВСЕГДА составляй подробный план действий и отправляй его на согласования.
8. НИКОГДА не вноси изменения в код без моего согласования.
9. Отвечай мне всегда на русском языке.
10. Никогда не пытайся самостоятельно запустить проект, даже для отладки.
11. Всегда следи за состоянием файла .gitignore
